About Lyuze

Lyuze is a feature-rich, self-hosted Discord bot designed to enhance your server experience. Built with modern technologies including Discord.Net for optimal performance and MongoDB for reliable data storage.

Lyuze offers a comprehensive suite of features including anime API integrations (Animechan for quotes, SauceNAO for image source detection, and Trace.moe for anime scene tracing), a leveling system with MongoDB integration, customizable user profiles with background settings and privacy controls, reaction roles for easy role management, and essential moderation commands including Purge, Kick, Ban, and Kill.

The bot is open-source and self-hosted, giving you complete control over your server's functionality. Whether you're building an anime-focused community or need robust moderation tools, Lyuze provides the features you need with the flexibility to customize to your server's requirements.
